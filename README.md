# Time

This is the git repository for all of my teaching materials related to the topic of Time. 

For Fall 2026, this includes all course materials for [190.426](190.426_TimePolitics.md), “Time and Politics,” as well as additional materials for 190.690, “Untimely Politics.”

All materials are available just by selecting the file name, but it may be easier to [**start here**](190.426_TimePolitics.md) and then follow the links provided. Students will want to follow *all* links and read *all* materials.
